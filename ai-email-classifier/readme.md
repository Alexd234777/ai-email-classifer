# AI Email Tone Classifier (Flask App)

This Space hosts a **full-stack Flask application** that analyzes message tone using
a hybrid NLP engine built with:

- Toxicity model
- Offensive language model
- Sentiment model
- Emotion model
- Rule-based overrides (threats, profanity, insults, sarcasm)
